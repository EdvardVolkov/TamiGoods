# TamiGoods - Online Marketplace

Modern online marketplace built with Next.js

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Production Build

```bash
npm run build
npm start
```

## Technologies

- Next.js 13
- TypeScript
- Tailwind CSS
- React 18
